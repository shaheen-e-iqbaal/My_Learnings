
-> process is defined as a "Instance of a program which is currently running."

-> Thread is a light weight process. it is a light weight unit of execution inside a process. single process can have more than one threads. each thread doesn't get separate RAM memory. they share the memory allocated to a process, whose they are part of.

-> Concurrency is defined as "Trying to do multiple task simultaneously."

-> Parallelism is defined as "Actually doing multiple task simultaneously." parallelism is special case of concurrency. parallelism can only be achieved if you have multiple core CPU. if multiple independent threads or processes runs at the same time on different CPU cores, then it is called that parallelism is achieved.
